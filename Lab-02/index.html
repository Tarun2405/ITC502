PROJECTS

Detection of animals using convolutionary neural network.

Vehicle animal collision is most common incident which causes loss to wild life. This intelligent system helps in reducing the roadkill during the active nocturnal hours. The system uses thermal image processing using the camera mount on the vehicle. It extracts the features of the object in front of the vehicle to detect the animal. The main method used to extract the features of the animals is VGG-16. The features are detected by normalizing the radiometric image and then processed by finding the magnitude and gradient of a pixel. The extracted features are given as an input to the basic deep learning model, a one-dimensional convolutional neural network, where binary cross-entropy is used to detect the existence of the object. Thus the animal can be detected.

Optimized Performance Medical Image Classification with Deep Convolutional Neural Networks.

In this study, we push towards setup a reproducible analysis pipeline to reveal the impact of deep learning techniques on medical image classification performance with deep convolution neural networks. By computing the performance of multiple deep learning techniques, we want to compare them to a baseline pipeline and, thus, identify possible performance gain. Furthermore, we explore the possible performance impact on multiple medical datasets from diverse modalities ranging from histology to X-ray imaging. Our experiments aim to help understand the beneficial as well as unfavorable influences of different deep learning techniques on model performance. This study contributes to the field of deep understanding and provides detection for ISIC & CMNIST histology cancer image classifications.
